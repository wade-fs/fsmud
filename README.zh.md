# 目標

## 近期

- 實作一個可以 telnet 的簡易環境。
	- 有想過找[TalesMud][3]這類的現成環境來修改，後來覺得沒意義，重新寫也是一種很好的學習過程。
- 具備簡易[MUD][7]的環境。

## 中期

- 實作 web UI 環境進行圖形虛擬世界。
- 實作跟[TalesMud][3]一樣具備 Mudlib 的完整遊戲世界。

## 長期

- 利用 AI 讓 MUD 中的人物具有智慧
- 提供地圖製作
- 可以的話具備3D效果，類似[minecraft][8]
